# About

I am a **GIS analyst and geospatial software developer** focused on building tools that connect **enterprise GIS systems, cloud geospatial infrastructure, and emerging GeoAI workflows**.

My work focuses on:

• ArcGIS Pro SDK development  
• Python geospatial automation  
• spatial ETL and QA/QC pipelines  
• geospatial web applications  
• cloud-native geospatial data systems  
• AI-assisted spatial tooling

Many of my projects explore how modern developer tooling and AI workflows can improve the durability and usability of GIS systems.

## FaultLine

Cloud-native geospatial processing framework.

FaultLine explores modern geospatial pipeline design for moving datasets between:

• enterprise GIS systems  
• cloud object storage  
• web applications  
• spatial processing environments

The project focuses on building reusable components for:

- spatial ETL
