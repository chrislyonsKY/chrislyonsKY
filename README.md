# .github

Community health files, shared workflows, documentation, and AI-development infrastructure for the **chrislyonsKY** GitHub organization.

## What's Here

| Folder | Contents |
|--------|----------|
| `profile/` | [Org profile README](profile/README.md) shown on [github.com/chrislyonsKY](https://github.com/chrislyonsKY) |
| `docs/plans/` | Ecosystem plans and roadmaps |
| `docs/catalogs/` | Package catalogs for PyPI, CRAN, npm, NuGet, crates.io, and QGIS |
| `docs/compliance/` | Registry publishing compliance guides and checklists |
| `docs/branding/` | Visual identity and package branding specs |
| `shared/test-fixtures/` | Language-agnostic test data (GeoJSON, YAML, JSON, GeoTIFF) |
| `shared/workflow-templates/` | Reusable GitHub Actions workflows for publishing |
| `.github/` | Copilot instructions and repo configuration |

## Registries

CRAN (R) · PyPI (Python) · plugins.qgis.org (QGIS) · crates.io (Rust) · NuGet (.NET) · npm (JS/TS)
